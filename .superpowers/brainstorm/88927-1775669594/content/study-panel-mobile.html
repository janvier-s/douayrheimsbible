A — Bottom Sheet

Study notes slide up from the bottom. Reading text stays visible above. Drag handle to expand/collapse. Tap a verse to jump notes to that verse.

  • Natural mobile gesture (familiar from maps, music apps)
  • Text always visible
  • Expandable for deep study
  • More implementation complexity

B — Full-screen overlay

Tapping "Study mode" opens study notes full-screen with a close button. Reading text is fully hidden but user can return instantly.

  • Maximum reading space for notes
  • Clean, focused experience
  • Simple to implement
  • Can't see text and notes together
  • Switching feels abrupt

C — Tab switcher

A tab bar below the top navigation switches between the reading text and the study panel. Both fill the full screen, toggled by a tab.

  • Simple mental model
  • Easiest to implement
  • Standard mobile pattern
  • No cross-reference while reading
  • Scroll position lost when switching tabs
